# port_scanner

Projet Python de scan de ports TCP.

## Fonctionnement
L'utilisateur saisit une adresse IP et une plage de ports.
Le programme teste les ports un par un et affiche ceux qui sont ouverts.

## Fichiers
- main.py : interface utilisateur
- scanner.py : logique de scan
- utils.py : validation des entrées

## Lancement
```bash
python main.py
